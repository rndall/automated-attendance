<h1 align="center">CCIS Attendance Recording System (CARS)</h1>

The **CCIS Attendance Recording System (CARS)** is designed to help event organizers and the student council of the CCIS Department accurately and efficiently record student attendance during events. It ensures error-free and reliable attendance tracking through automated recording and verification features. This project was initiated by **CCIS Dean Sir George Granados** and developed by the **Information and Communications Technology Department Unit**.

## Requirements
- **Node.js**: https://nodejs.org/en/download
- ***Camera**

## Installation

1. Install dependencies
```
npm i
```

or

```
npm install
```

2. Run the system
```
npm run dev
```
